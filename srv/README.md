## 关于外键  

- 外键会对增删改有影响，但是对查询没影响；  
- 外键并不会影响性能，考虑分开查询的I/O 问题，但是会较长时间地`锁表`从而影响并发；  
- 不论用不用外键，业务并不会减少；  
- 开始时用外键约束，随着业务扩展，逐步迁移到应用层；  

### 推荐使用  

1. 数据量不大  
2. 系统安全性高  

### 避免使用  

1. 数据量极大  
2. 高并发
